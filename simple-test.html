Teste Simples da Agenda

Passo 1: Fazer Login

1. Abra http://localhost:5173

2. Use email: admin@dudufisio.com

3. Use senha: demo123456

Passo 2: Navegar para Agenda

1. Após o login, clique em "Agenda" no menu lateral

2. Ou vá diretamente para http://localhost:5173/agenda

Passo 3: Verificar se a Agenda Aparece

✅ Deve mostrar um calendário semanal

✅ Deve mostrar agendamentos coloridos

✅ Deve mostrar nomes dos terapeutas no topo

❌ Se estiver em branco, há um problema no carregamento dos dados

Debug Manual

Se a agenda estiver em branco:

  1. Abra o DevTools do navegador (F12)
  2. Vá para a aba Console
  3. Procure por erros em vermelho
  4. Vá para a aba Network
  5. Recarregue a página e veja se alguma requisição falha
  6. Na aba Console, digite: console.log('React version:', React.version)
  7. Na aba Console, digite: console.log('App state:', window)
